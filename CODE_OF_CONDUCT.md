# Code of Conduct

This project expects respectful, professional collaboration.

## Expected Behavior

- Be direct and constructive.
- Focus criticism on code, design, documentation, and maintainability.
- Assume good intent, but ask for clarification when something is unclear.
- Respect different levels of experience and different technical backgrounds.
- Keep discussions relevant to the project.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Discriminatory language or behavior.
- Publishing private information without permission.
- Repeatedly derailing technical discussions.
- Spam, trolling, or intentionally disruptive behavior.

## Reporting

If you see behavior that violates this code of conduct, report it to the project
maintainer. Include links, screenshots, or logs when available.

Reports will be reviewed privately. Maintainers may remove comments, close
issues, block users, or take other action needed to protect the project.

## Scope

This code of conduct applies to project spaces, including issues, pull requests,
discussions, documentation, and related community channels.
